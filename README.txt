README.txt
==========

What is this module?
--------------------

This module allow to add 3rd-party tracking code for digital marketing.

What can be integrated?
--------------------

1. PRIVY
2. GOOGLE ADWORDS REMARKETING
3. FACEBOOK PIXEL
4. HUBSPOT
5. KLAVIYO
6. ADROLL

INSTALLATION
------------

 * Go to /admin/modules and enable the Marketing Digital Integrations module.
 * Go to /admin/config/services/digital_integrations and click on checkboxes that you want to integrate.
 * Add code for api in text field.
 * Click "Save configuration".
